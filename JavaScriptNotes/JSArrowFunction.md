# 箭头函数
提供一种快速简洁地定义函数的方式；
```
    x => x*x; //等价于
    function (x):{
        return x*x;
    }
```
箭头函数相当于匿名函数，可以看出x直接表示函数的参数，箭头后方就表示函数的返回值；
```
    //函数体包含多条语句
    x => {
        if(x>0){
            pass;
        }else {
            pass;
        }
    }
    // 函数包含多个参数
    (x,y) => x * y;
    // 函数的返回值类型为对象
    x => ({'foo':'bar'});
```
